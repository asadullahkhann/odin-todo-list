# Todo List App

A simple todo list application that lets you create projects and manage todos within each project. Todos can be added, edited, deleted, and are saved in your browser and stay even after refresh.

## Features

* Create and manage multiple projects
* Add todos to each project
* Edit existing todos
* Delete todos
* Data is saved in the browser and persists after refresh

## Built With

* HTML
* CSS
* JavaScript
* Local Storage API

## Usage

1. Create a project.
2. Add todos inside the project.
3. Edit or delete todos as needed.
4. Your data is automatically saved in the browser.
